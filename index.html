Python vs. Java

Feb 24, 2019

Hey Everyone!

This weekend I tried to jump into Python a bit more and continue to familiarize myself with its syntax, functions, and methods. The first language I learned to program in was Java and so far, the differences between the two are pretty apparent. Many people told me that Python is a much easier language. It has capabilities such as being able to assign any data type to a variable, assigning multiple data types to a list, etc.

So far though it has been tough for me (granted this is only my second real language, I'm not counting HTML and CSS). The generality and ease of use that Python offers is, strangely enough, working against me so far. One thing I am finding particularly difficult is getting my loops to work correctly. The loop I used most commonly in Java was a basic for loop: for int i = 0; i < string.length(); i++. However Python tends to use while and for each loops much more. This makes me a bit confused as I'm finding it harder to work loops through strings or lists from index to index. The specificity that Java required made me start my learning having certain assumptions about data types and how code works and its all coming crashing down.

Now, on one hand, this is interesting for me. Starting to realize the pros and cons of one language over another makes me confident that in the future I will be able to select "the right language for the job" and design my software creations in the most effective and clean possible way. But it does feel a bit frustrating to know what to do to solve a problem but not how to phrase it. But the syntax is coming to me slowly and I'm sure it will be easier than Java as everyone says.

On the other hand though I made some big leaps in my CSS and HTML abilities this weekend and tidied up the site quite a bit in my opinion. I'm evaluating right now whether I want to make a mini web server using a RaspberryPi (which sounds super fun and like a good learning oppurtunity) to host the site, or hosting it through a service. Either way its an exciting and somewhat frightening thought. Who knows? Maybe someone will actually be reading these posts soon!


My First Blog Entry!

Feb 18, 2019

Hello, World!

I am super excited to finally have my own site to showcase my (currently nonexistent) portfolio, and share my numerous struggles and less numerous successes with anyone bored enough to listen!

A bit about me: I am a 22 year old techie from Boston, MA. As a typical New Englander I love iced coffee, the New England Patriots, Fall, and pretending I have no social skills! Outside of programming, development, coding I enjoy weightlifting (which I sometimes participate competitively in) playing videogames (totally digging Apex Legends right now), and overusing parentheses!

As to my coding experience I started coding about six months ago by taking a class at my local community college in Java, and quickly learned to love it. The applied problem solving and logical exercises were something that I excelled at and I quickly tried to teach myself everything about everything. Thus, I ended up falling into the same trap many new programmers do: wanting to learn but not knowing what to study and where to start. Quickly I realized that Java alone, although a valuable tool in my toolbelt, would not get my into a coding career as I had envisioned.

And that brings me to today! This website is my first attempt at front end development with HTML/CSS (still avoiding JavaScript at the time of this writing) and is my first official assignment for my coding course at Coding Temple! This program will be an intense 10 weeks and I can't wait to begin! Hopefully by the time I make my next post I'll figure out how to get the comment box working. Until then!
